## Planned

### AI pre-generation analysis (multi-course detection)
Перед генерацией курса анализировать загруженные материалы и определять,
следует ли создать один общий курс или несколько отдельных курсов по ролям/темам.

Пример: если загружены «Регламент для менеджеров» и «Инструкция для разработчиков»,
система предложит создать два курса вместо одного общего.

Подход:
1. POST /courses/analyze-materials — возвращает предложенную структуру курсов
2. Фронтенд показывает preview: сколько курсов будет создано, для кого
3. Owner подтверждает — запускается несколько generate-job параллельно

Статус: **не внедрять до согласования**
