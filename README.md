# Intern AI — корпоративное обучение на AI

Платформа генерирует персонализированные курсы для стажёров на основе внутренних материалов компании (документы, регламенты, аудио/видео). Внутри: AI-ментор с RAG по материалам, тесты с анализом слабых мест, доучивание, учёт затрат на OpenAI по компаниям.

## Стек

- **Бэкенд** — FastAPI + SQLite (`backend/`), OpenAI API (генерация, эмбеддинги, Whisper).
- **Фронтенд** — Next.js 15 + Tailwind (`frontend/`).

## Запуск

### 1. Бэкенд

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

API поднимется на `http://localhost:8000`. Схема SQLite создаётся автоматически при первом старте.

### 2. Фронтенд

```bash
cd frontend
npm install
npm run dev
```

Приложение — на `http://localhost:3000`.

## Переменные окружения

| Где | Переменная | Что это |
|---|---|---|
| `backend/.env` | `OPENAI_API_KEY` | Ключ OpenAI (обязательно) |
| `frontend/.env.local` | `NEXT_PUBLIC_API_URL` | Адрес бэкенда (по умолчанию `http://localhost:8000`) |

## Как пользоваться

1. Зарегистрируйтесь без кода приглашения — создастся компания, вы станете её владельцем.
2. Загрузите материалы компании на странице «Материалы».
3. Нажмите «Анализировать материалы» — AI предложит один или несколько курсов.
4. Запустите генерацию; пригласите сотрудников по коду из админки и назначьте им курсы.

Подробности архитектуры — в [CLAUDE.md](CLAUDE.md).
